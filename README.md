To-Do List Web App Sederhana

Aplikasi web sederhana untuk mencatat daftar kegiatan harian.  
Dibuat menggunakan HTML, CSS, dan JavaScript.

Tujuannya adalah membantu pengguna mengelola kegiatan dengan mudah
langsung dari browser.

---

Fitur Utama
- Menambah dan menghapus tugas.
- Validasi input agar tidak kosong.
- Data tersimpan sementara di browser.
- Desain responsif untuk mobile dan desktop

---

Cara Pakai
1. Ketik tugas di kolom input.
2. Tekan tombol *Tambah* untuk memasukkan ke daftar.
3. Klik tugas yang sudah selesai untuk menandainya.

---

Struktur Folder & Teknologi
*Teknologi:*
- HTML5 (struktur halaman)
- CSS3 (layout dan styling responsif)
- JavaScript (DOM manipulation dan event handling)
